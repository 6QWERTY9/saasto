# Saasto
---
# Описание проекта
---
Saasto — лендинг-страница для приложения по отслеживанию времени. Проект на React, TypeScript, SCSS с архитектурой FSD. 
**Макете взят из открытых источников и используется только в учебных целя.**

## Стек технологий
* React 18+ (хуки)
* TypeScript
* SCSS Modules
* Vite
* ESLint, Prettier

---

## Установка и запуск

1. Клонируйте: `git clone https://github.com/your-username/saasto.git`
2. Установите: `pnpm install`
3. Запуск: `pnpm run dev`
4. Сборка: `pnpm run build`

---

## Структура проекта

```
src/
├── app/          # Глобальные настройки
├── pages/        # Страницы (лендинг)
├── features/     # Фичи (подписка, демо)
├── shared/       # Общие компоненты/утилиты
└── widgets/      # Виджеты (хедер, футер)

```

---

## Скрипты

* `pnpm run dev` — разработка
* `pnpm run build` — продакшен
* `pnpm run lint` — линтинг
